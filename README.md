# Personal Site

My xiaofengyuan.com site based on Jekyll.

## ENV Setup

```bash
# Jekyll setup
bundle install

# Lint setup
yarn
brew install yamllint autocorrect
```

## Lint

```bash
# Lint all
yarn lint

# Fix lint issues
yarn fix
```
